# 演示视频脚本（1–2 分钟）

目标：完整展示"拨打电话 → 自然对话采集 → 保安企业微信收到 → 确认放行"，并体现 ≤25 秒。

## 拍摄准备
- 一部手机拨打 Twilio 号码；另一屏显示企业微信群 + web 终端日志。
- 秒表/屏幕计时从 **Agent 开口** 开始。

## 分镜

| 时间 | 画面 | 旁白（可选） |
|---|---|---|
| 0:00 | 手机拨打贴在"入口"的号码，接通 | "未登记车辆，拨打入口电话" |
| 0:03 | Agent 开口（开始计时）："您好，请问车牌号多少，今天找哪家公司，什么事儿？" | "AI 门卫一句话问全" |
| 0:06 | 访客："沪A12345，来蓝色鲸鱼送货的" | "一句话同时给多项，不机械" |
| 0:10 | Agent："好嘞，手机号方便留一下吗？" → 访客："138……" | "只补缺的那项" |
| 0:16 | Agent 复述："好的！沪A12345，蓝色鲸鱼送货，已通知门卫，请稍等放行。" | |
| 0:18 | 切企业微信群：访客卡片弹出（停计时，显示 < 25s） | "结构化信息已到保安手机" |
| 0:22 | 保安点"✅确认放行"→ 浏览器"已放行 ✓"，终端打印 [GATE] 抬杆指令 | "确认即放行，生产对接海康 ISAPI" |
| 0:28 | （加分）口播："同一车牌再来会被识别为回访，直接确认；保安还能自然语言查数据" | |

## 加分镜头（可选，再 30 秒）

- 终端跑 `python -m visitor_agent.guard_query "什么时间段访问最多？"` 展示查询。
